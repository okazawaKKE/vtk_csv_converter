
Manual readFOAM.py

・ ./src/readFOAM.py
可視化などの処理をするためのライブラリ

・./src/rotation_df.py
dataframeの回転座標系対応のためのライブラリ

・ ./convert.py
実行用メインスクリプト
カレントディレクトリのopenfoamの結果フォルダを読み込み可視化する

・ ./settings.py
convert.pyを実行する際に使う、パスや引数などの記述
最初に可視化のオプションをここで設定する


***必要な外部ライブラリ***
・matplotlib
・numpy
・vtk
・pandas
・scipy
